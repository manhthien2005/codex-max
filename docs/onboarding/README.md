# Onboarding Notes

This directory is reserved for repository onboarding artifacts produced by the curated `codebase-onboarding` workflow.

## Suggested contents

- stack summary
- architecture map
- key entry points
- directory roles
- common commands
- request lifecycle notes
- project-specific conventions

## Conventions

- Keep onboarding notes short and scannable
- Prefer one onboarding note per target repository or module
- Update the note when major structure or tooling changes
